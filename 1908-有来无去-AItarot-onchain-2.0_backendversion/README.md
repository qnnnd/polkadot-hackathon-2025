## 项目简介
Altarot OnChain 2.0 是一个基于区块链的塔罗牌 AI 占卜 & NFT 铸造 DApp，支持链上订单、AI 解读、NFT 铸造与排行榜功能，兼容多链和自托管后端。

## 技术栈
- 前端：React, ethers.js
- 后端：Node.js, caddy
- 智能合约：Solidity (ERC721)
- 存储：IPFS, Pinata
